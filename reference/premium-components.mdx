## Premium Components

Our premium documentation includes enhanced UI components that provide a superior user experience with improved visual hierarchy and interactive elements.

## Premium Callout

The premium callout component highlights important information with distinctive styling:

```html
<div class="premium-callout">
  <span class="premium-badge">PREMIUM CONTENT</span>
  <h3>Component Title</h3>
  <p>Description of the premium content or feature.</p>
</div>
```

## Custom Tailwind Extensions

We've extended Tailwind with custom color definitions and spacing utilities that align with our brand guidelines:

```js
// tailwind.config.js
theme: {
  extend: {
    colors: {
      'gs-brand-blue': '#1e40af',
      'gs-brand-slate': '#334155',
      'gs-border-muted': 'rgba(51, 65, 85, 0.2)',
    },
    boxShadow: {
      'gs-chrome': '0 14px 34px rgba(30, 64, 175, 0.12)',
    },
    spacing: {
      'vertical': 'clamp(2.5rem, 2rem + 1vw, 3.5rem)',
    },
  },
}
```

These extensions ensure consistent styling across all components while maintaining flexibility for future enhancements.
