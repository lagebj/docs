## Purpose

The Design System provides a reusable set of UI patterns, components, and styling rules that ensure a cohesive experience across the Mintlify documentation site. It is built on top of **Tailwind CSS** and extends it with our brand colors, spacing, and shadows.

## Core Foundations

| Token | Value | Tailwind Class |
|-------|-------|----------------|
| Primary Blue | `#1e40af` | `text-gs-brand-blue` |
| Slate | `#334155` | `text-gs-brand-slate` |
| Border Muted | `rgba(51, 65, 85, 0.2)` | `border-gs-border-muted` |
| Shadow | `0 14px 34px rgba(30, 64, 175, 0.12)` | `shadow-gs-chrome` |
| Vertical Spacing | `clamp(2.5rem, 2rem + 1vw, 3.5rem)` | `spacing-vertical` |

## Component Library

### Callout (Premium)
```html
<div class="premium-callout">
  <span class="premium-badge">PREMIUM CONTENT</span>
  <h3>Component Title</h3>
  <p>Description of the premium content or feature.</p>
</div>
```
- **Usage** – Highlight important information. Apply `bg-gs-brand-slate` for background and `border-gs-border-muted` for an outline.

### Button
```html
<a href="#" class="px-4 py-2 rounded bg-gs-brand-blue text-white hover:bg-gs-brand-blue/90 transition">Get Started</a>
```
- **Variants** – Primary (blue), Secondary (slate), Ghost (transparent with border).

### Navigation Bar
- Fixed top, uses `bg-white` with `shadow-gs-chrome`.
- Logo on the left (`logo-navbar.svg`).
- Nav items use `text-gs-brand-slate` and `hover:text-gs-brand-blue`.

## Layout & Grid

- **Container** – `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`.
- **Vertical Rhythm** – Use the `spacing-vertical` utility for consistent vertical gaps between sections.

## Accessibility

- All interactive components must be focusable (`tabindex`), have discernible text, and meet WCAG AA contrast.
- Use `aria-label` for icons without visible text.

## Contribution

- Add new components in `docs/reference/` and update `tailwind.config.js` accordingly.
- Follow the same front‑matter structure (`--- title: … description: … ---`).
