# Paperclip Operating Doctrine

This package contains the core principles, practices, and guidelines for operating Paperclip according to Grounded Systems principles.

## Structure

- **principles** - Core foundational principles that guide all Paperclip operations
- **practices** - Concrete practices and procedures for day-to-day operations
- **decision-frameworks** - Frameworks for making consistent decisions across teams
- **anti-patterns** - Common mistakes and pitfalls to avoid
- **guidelines** - Specific operational guidelines for various scenarios
