# Grounded Systems

This repository contains the Mintlify documentation site for **Grounded Systems**.

## Purpose

The site serves two roles:

1. Public entrypoint for potential customers and future employees
2. Knowledge base for the team itself

## Structure

- `customers/` public customer-facing entry pages
- `practitioners/` public practitioner-facing entry pages
- `principles/` core stance and non-negotiables
- `playbook/` actionable field guidance
- `cases/` examples and scenarios
- `reference/` glossary, FAQ, and contribution guidance

## Local development

Install dependencies and run Mintlify locally.
Use an LTS Node.js version (Mintlify does not support Node 25+).

```bash
npm install
npm run dev
```
